# SIIG Financial Formulas

## Net Worth
Net worth is the difference between total assets and total liabilities.

Formula:
```
Net Worth = Total Assets - Total Liabilities
```

Example:
- Assets = 2,500,000 THB
- Liabilities = 1,200,000 THB
- Net Worth = 1,300,000 THB

## Savings Rate
Savings rate measures how much of monthly income is saved.

Formula:
```
Savings Rate = (Monthly Savings / Monthly Income) × 100%
```

Where:
- `Monthly Savings = Income - Expenses`

Example:
- Income = 100,000 THB
- Expenses = 70,000 THB
- Savings Rate = ((100,000 - 70,000) / 100,000) × 100% = 30%

## Debt Service Ratio (DSR)
DSR measures the share of income used to service debt payments.

Formula:
```
DSR = (Total Monthly Debt Payments / Monthly Income) × 100%
```

Example:
- Monthly debt payments = 25,000 THB
- Monthly income = 120,000 THB
- DSR = (25,000 / 120,000) × 100% = 20.8%

## Emergency Fund
Emergency fund target is typically 3–12 months of essential expenses.

Formula:
```
Emergency Fund = Monthly Essential Expenses × Months of Coverage
```

Common ranges:
- 3 months = conservative short-term safety
- 6 months = baseline for stable households
- 9–12 months = robust protection

Example:
- Essential expenses = 40,000 THB
- Coverage = 6 months
- Emergency Fund = 240,000 THB

## Goal Future Value (Goal FV)
Goal future value calculates how much a savings target will grow to by a future date.

Formula:
```
FV = PV × (1 + r)^n
```

Where:
- `PV` = present value or current balance
- `r` = expected periodic return rate
- `n` = number of periods

Example:
- PV = 50,000 THB
- r = 5% per year
- n = 5 years
- FV = 50,000 × (1.05)^5 = 63,814 THB

## Goal Monthly Payment (Goal PMT)
Goal PMT calculates the monthly deposit required to reach a target amount.

Formula for regular monthly contributions with compound interest:
```
PMT = (FV - PV × (1 + r)^n) × r / ((1 + r)^n - 1)
```

Where:
- `FV` = future goal amount
- `PV` = current savings balance
- `r` = monthly interest rate
- `n` = number of months

Example:
- FV = 500,000 THB
- PV = 50,000 THB
- Annual rate = 6% → monthly r = 0.06 / 12 = 0.005
- n = 60 months
- PMT ≈ (500,000 - 50,000 × 1.005^60) × 0.005 / (1.005^60 - 1)
- PMT ≈ 6,876 THB

## Retirement Nest Egg
Retirement nest egg estimates the capital needed to support retirement income.

Formula (simple multiplier):
```
Retirement Nest Egg = Annual Retirement Income × Withdrawal Factor
```

Common factor range:
- 20–25 × annual income for a 4–5% safe withdrawal rate

Example:
- Desired annual income = 1,200,000 THB
- Factor = 25
- Nest Egg = 30,000,000 THB

Alternative formula using replacement ratio:
```
Retirement Nest Egg = Current Income × Replacement Rate × Retirement Years
```

## Insurance Need
Insurance need is the amount required to protect dependents and replace lost income.

Basic formula:
```
Insurance Need = Debt + Future Income Need + Education Cost + Emergency Reserve - Existing Coverage
```

Common components:
- Outstanding liabilities (mortgage, loans)
- Future income replacement (income × years remaining)
- Education or legacy costs
- Emergency reserve buffer
- Less existing life coverage and cash savings

Example:
- Debt = 3,000,000 THB
- Income need = 500,000 THB × 10 years = 5,000,000 THB
- Education reserve = 1,000,000 THB
- Existing coverage = 2,000,000 THB
- Insurance Need = 7,000,000 THB
